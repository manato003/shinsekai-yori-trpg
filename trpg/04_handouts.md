## HO1 ── 渡辺早季（わたなべ さき）

**配布対象**: HO1担当PL

---

### キャラクター基本情報

| 項目 | 情報 |
|---|---|
| 氏名 | 渡辺 早季（わたなべ さき） |
| 年齢 | 12歳（第1・2話）/ 26歳（第3話） |
| 呪力形態 | **視覚共有** ＋ **念動力（テレキネシス）** |
| 職業 | 全人学級の生徒 → 神栖66町・異類管理課 |

**呪力の特徴**:
視覚共有は「離れた場所の景色を自分の目でリアルタイムに見る」能力。
偵察・情報収集に絶大な力を発揮する。念動力と組み合わせると、
「見えない場所に力を及ぼす」という強力な応用が可能。

---

### 世界観ブリーフィング（PLが知っている情報）

あなたは神栖66町で生まれ育った。この町は「呪力」と呼ばれる超能力を持つ人々だけが住む、
穏やかで美しい場所だ。子供たちは和貴園で育ち、全人学級で呪力の訓練を受ける。

呪力を持つ人間は神に近い存在であり、「バケネズミ」と呼ばれる知的生物が我々に仕える。
世界は秩序正しく、完璧に見える。

ただし——子供の頃から、一つだけ引っかかることがある。

---

### 秘密（このPLだけが知る情報）

> **以下はHO1のPLだけが読んでください**

「姉がいた」という感覚がある。

でも誰もそんな姉を知らないし、家族の写真にも姉は映っていない。
両親に聞くと「一人っ子よ」と笑われる。

夢の中で、誰かが拙い文字で「ごめんなさい」と書くのを見た。
誰かが、消えていった。あなたの記憶から。

**動機**: 仲間を守りたい。でも何かが、ずっと引っかかっている。

**SANボーナス**: 「仲間が無事だと確認できたとき」SAN +1D3

**フラグ**:
- 第1話: ミノシロモドキの「処分の基準」を聞いたとき、「姉がいた」という感覚が確信に変わる
- 第2話: 不浄猫を追って瞬のもとへ向かう。瞬との別れを経験する
- 第3話: 真理亜・守の死亡確認。母の遺書を読む。スクィーラの処遇を最終的に決断する

---

### 推奨技能の振り方ガイド

**職業技能点**: EDU × 2 + POW × 2（例: EDU70・POW65なら210点）

| 技能 | 推奨割り振り | 理由 |
|---|---|---|
| **呪力制御** | 50〜70% | メイン能力。視覚共有の安定運用に必須 |
| 目星 | 40〜60% | 仲間の異変・周囲の危険を最初に察知する役割 |
| 心理学 | 40〜60% | NPCの嘘・仲間の本音を読む |
| 聞き耳 | 35〜50% | 察知・情報収集の補助 |
| 応急手当 | 40% | 仲間を救う場面で活躍 |
| サバイバル | 30〜40% | 八丁標の外での生存判定 |

**趣味技能点**: INT × 2（例: INT80なら160点）
→ 自然知識・水泳・隠れる など自由に割り振ってよい

---

### このキャラで楽しむコツ

**「気づいてしまう」主人公を演じる。**
早季は仲間の中で最も「何かがおかしい」と感じる感受性を持つ。
呪力の偵察能力で「見てしまう」ことが物語を動かす。
鈍感に演じるより、**不安を感じながらも行動する**ヒロインとして演じると物語が締まる。

## 愧死機構フロー

```
PC が「人間」に対して呪力で殺意ある攻撃を試みる
        ↓
   即座に POW × 5 判定（愧死機構の発動抵抗）
        ↓
ファンブル → 即死（意識する間もなく）
失  敗  → HP -2D10（臓器破壊）。0以下で即死
成  功  → 今回は免れる。SAN -1D4（精神的苦痛）

※注意: 「悪鬼（真理亜の子）」に対して呪力を向けると
      悪鬼が「バケネズミ」と認識しているため、
      「バケネズミを攻撃 = 同種攻撃」と誤認識し、
      PCに愧死機構が発動してしまう！
```

**愧死機構が発動しない例外**:
- バケネズミへの攻撃（同種と認識しない）
- 人間を物理的に攻撃（武器・素手）
- 意図せず人間に呪力が当たった場合（KP裁量でSAN損失のみ）

---

## 呪力使用フロー

```
呪力を使う → POW × 5 判定
  （呪力制御技能で命中ボーナスダイス追加可）
        ↓
クリティカル → 通常の2倍の効果
成  功  → 意図した効果が発動
失  敗  → 呪力が空振り。MP -1
ファンブル → 制御を失い、意図しない効果が周囲に発生

真言を唱えた: ボーナスダイス +1
真言を忘れた・唱えられない: ペナルティダイス -1
```

**MP管理**:
- 最大MP = POW ÷ 5
- MP消費目安: 軽作業1MP / 中作業2MP / 重作業3〜5MP
- MP 0 → 呪力使用不可
- 回復: 睡眠1時間で MP+1
