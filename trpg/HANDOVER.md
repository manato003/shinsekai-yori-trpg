## 各ファイルの役割

### `00_rules.md` — 全セッション共通参照
CoC 7版に乗せたカスタムルール。**台本を修正・追加するときは必ずここを確認する。**

核心ルール:
- **呪力**: POW×5判定、真言でボーナスダイス、MP消費
- **愧死機構**: 人間への呪力攻撃 → POW×5判定、失敗でHP-2D10（即死あり）
- **業魔エリア**: 進入でSAN-1D6、1R留まるごとにCON×5判定
- **悪鬼**: 呪力攻撃不可（愧死機構が逆発動）→ ギミック戦闘のみ

### `01_session1.md` — 第1話「偽りの箱庭」
12歳編。主要シーン: キャンプ → ナイトカヌー → ミノシロモドキ → 呪力凍結 → 土蜘蛛 → 脱出

### `02_session2.md` — 第2話「欠けゆく輪」
14歳編。**瞬（HO5）が前半で業魔化・消滅、真理亜（HO3）・守（HO4）が中盤離脱確定。**
PLへの事前告知タイミングが台本内に明記されている。

### `03_session3a.md` + `03_session3b.md` — 第3話「業火の獣たち」
26歳編。第3話はトークン制約で前後半に分割:
- 3a: 夏祭り奇襲 → 鏑木の死 → 東京地下探索 → サイコ・バスター発見
- 3b: 悪鬼最終決戦（奇狼丸の囮作戦） → スクィーラ裁判 → マルチエンディング

### `04_handouts.md` — 補助資料
- プレイヤーHO×5（キャラ説明・秘密・推奨技能）
- クリーチャーシート（不浄猫・土蜘蛛・悪鬼・スクィーラ・奇狼丸など）
- GMスクリーン用クイックリファレンス
- キャンペーン全体の伏線・回収チェックリスト

## 注意事項・既知の制約

**トークン管理**:
- `shinsekaiyoriGeminiMatome.txt`（18KB）は全セッションで使える（約5,000トークン）
- OCR全文 `shinsekaiyori.md`（2MB）は単体で200kトークン超 → 直接読み込み不可
- `source/` 以下の分割ファイルを使うこと（最大742KB）

**HO確定離脱キャラ**:
- 瞬（HO5）: 第2話前半で業魔化・消滅 → PLには第2話開始前に告知
- 真理亜（HO3）・守（HO4）: 第2話中盤で離脱 → PLには第2話開始前に告知
- 離脱後のPLは奇狼丸・乾などの代替キャラを担当（04_handouts.md に記載）

**愧死機構の運用**:
- 「殺意を持った呪力」の判断はKP裁量 → KPが迷ったら「防衛目的か攻撃目的か」で判断
- 悪鬼への呪力攻撃は愧死機構が**発動してしまう**（最終決戦の核心ギミック）

**システム**:
- CoC 7版（クトゥルフ神話TRPG 第7版）
- 呪力はカスタムマジックシステム（既存の魔術・呪文システムとは別扱い）
