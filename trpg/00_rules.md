# 新世界より CoC 7版 カスタムルールシート

**システム**: クトゥルフ神話TRPG 第7版（CoC 7）
**キャンペーン構成**: 全3話（12歳編・14歳編・26歳編）
**想定プレイ人数**: KP1名 + PL4〜5名

---

## 1. 世界概要（KP向け）

舞台は西暦3000年頃の日本・神栖66町。人類は「呪力（PK）」という念動力を獲得したが、
その暴走を防ぐため社会全体が徹底した管理・記憶操作・暗殺によって秩序を維持している。
プレイヤーはその社会の内側に生きる少年少女として、少しずつ「世界の真実」に近づく。

**ゲームの核心**: 恐怖は外敵ではなく「自分たちが信じていた社会そのもの」にある。
CoCの狂気システムはこの設定と完璧に一致する。

**職業技能点**: EDU × 2 + POW × 2

## 3. 呪力（PK）システム

### 3-1. 呪力とは

人間の想像力を現実に投影する無限の力。念動力、炎操作、水操作、視覚共有など、
使用者の「心の形」によって発現する形態が異なる。

### 3-2. 呪力の使用

**基本判定**: POW × 5（呪力制御技能でボーナスダイス可）

| 結果 | 効果 |
|---|---|
| 成功 | 意図した効果が発動 |
| 失敗 | 呪力が空振り、MP -1 |
| ファンブル | 制御を失い、意図しない効果が周囲に発生 |
| クリティカル | 通常の2倍の効果 |

**MP消費**: 難易度に応じて1〜5MP。MPが0になると呪力使用不可。
睡眠1時間でMP +1（最大MP = POW ÷ 5）。

### 3-3. 真言（マントラ）

呪力の発動を安定させる自己暗示の呪文。PCごとに固有の真言を設定する。

- **真言を唱えた場合**: 呪力制御判定にボーナスダイス1個
- **真言を忘れた・唱えられない場合**: ペナルティダイス1個
- **呪力凍結状態（離塵などによる妨害）**: 真言を思い出す判定（INT × 5）に成功すると呪力が解放される

### 3-4. 呪力の形態例

| 形態 | 効果例 |
|---|---|
| 念動力（テレキネシス） | 物体を動かす、投擲する（ダメージ: STR相当） |
| 炎操作 | 出火させる、炎を制御する（1D6〜3D6ダメージ） |
| 水操作 | 水流を操る、水中で呼吸を確保する |
| 視覚共有 | 離れた場所の視覚情報を得る（偵察） |
| 物質操作 | 岩・土・金属を変形させる（STR×2相当） |
| 治癒呪力 | HP +1D4回復（MP 2消費） |

## 4. 愧死機構（きしきこう）

これが本シナリオ最大の制約ルール。

### 4-1. 定義

人類の呪力保持者は遺伝子レベルで「攻撃抑制」と「愧死機構」を埋め込まれている。
**人間（呪力使いの同種）に対して殺意を持って呪力を使おうとすると、
腎臓・副甲状腺が無意識の呪力によって自動的に機能停止し、死に至る。**

### 4-2. ゲームルール

PCが「人間を呪力で直接攻撃しようとする」場合、以下が自動発生する：

1. **即座にPOW × 5判定**（愧死機構の発動抵抗）
   - **失敗**: HP -2D10（臓器破壊）。ゼロなら即死
   - **成功**: 今回は免れるが、SAN -1D4（精神的苦痛）
   - **ファンブル**: 即死（意識する間もなく倒れる）

2. **「殺意を持って」の定義はKPが判断**。防衛的な行動（相手の武器を弾く、拘束する）は対象外。

3. **悪鬼（真理亜の子）に対して**: 悪鬼は「自分がバケネズミ」と思っているため、呪力を向けると愧死機構が**発動してしまう**（バケネズミを攻撃 = 同種攻撃と誤認識）。これが最終決戦の核心ギミック。

### 4-3. 例外

- バケネズミへの攻撃: 愧死機構は発動しない（同種と認識しない）
- 人間を物理的に攻撃（武器・素手）: 愧死機構は発動しない（呪力限定の制約）
- 意図せず人間に呪力が当たった場合: KP裁量でSAN損失のみ

## 5. 業魔化ルール（橋本・アッペルバウム症候群）

### 5-1. 定義

呪力が無意識に漏れ出し止まらなくなる病。発症した者は自分の意思に関係なく、
周囲の生物・物体・空間を歪め、異形化させてしまう。

### 5-2. 発症条件（PCには原則適用しない / NPCギミック）

- SAN値が最大値の1/5以下になった状態で呪力を大量消費した場合
- KPの判断でNPC（瞬）に適用する特殊展開

### 5-3. 業魔エリアのルール

業魔化した者の周囲**数十メートル**は「汚染エリア」となる。

| 汚染エリア内での行動 | ルール |
|---|---|
| 入った瞬間 | SAN -1D6（目撃判定） |
| 1ラウンド留まるごとに | CON × 5判定。失敗でHP -1D4（組織の変形） |
| 業魔本人に近づく | CON × 3判定。失敗でHP -1D6 |
| 業魔本人と会話する | 可能（距離10m以内、接触なし）。SAN -1D3 |

---

## 6. 悪鬼（ラーマン・クロギウス症候群）

### 6-1. 定義

攻撃抑制と愧死機構が**完全に欠如**した人間。
呪力で無制限に殺戮を行える存在。通常の人間の呪力攻撃は愧死機構のせいで逆に自分を傷つける。

### 6-2. 戦闘ルール（直接戦闘は不可能）

**呪力による直接攻撃**: PCが悪鬼に呪力攻撃を試みると、愧死機構ルール（4-2）が適用される。
→ 事実上、正面戦闘は**自殺行為**。

**物理攻撃**: 可能だが、悪鬼も呪力を持つため、飛来する物体は全て弾かれる（DEX × 5で回避）。

**勝利条件**: 悪鬼の「思い込み」を利用したギミック戦闘のみ。
具体的には第3話の「奇狼丸の囮作戦」参照。

### 6-3. 悪鬼のステータス（参考）

| ステータス | 値 |
|---|---|
| STR / CON / SIZ | 55 / 70 / 40 |
| DEX / INT / POW | 65 / 55 / 90 |
| HP | 14 |
| MP | 18 |
| 装甲 | なし（ただし呪力バリア: 受ける物理ダメージを1D6軽減） |
| SAN | 適用なし（愧死機構なし） |

## 7. SAN（正気度）チェック一覧

本シナリオで頻繁に発生するSANチェックの基準表。

| 状況 | 失敗時SAN損失 | 成功時SAN損失 |
|---|---|---|
| ミノシロモドキから歴史の真実を聞く | 1D10 | 1D3 |
| 不浄猫（暗殺猫）と遭遇する | 1D6 | 1 |
| 処分された子供の「存在の消去」を目撃/実感する | 1D8 | 1D3 |
| 業魔化した瞬のバンガローに入る | 1D6 | 1 |
| 瞬との最後の別れ | 1D6 | 1D3 |
| 真理亜・守の手紙を発見する | 1D4 | 1 |
| 夏祭りの奇襲・大量虐殺の目撃 | 1D8 | 1D3 |
| 鏑木（最強の呪力使い）が悪鬼に惨殺される目撃 | 1D10 | 1D6 |
| 東京地下の異形クリーチャーとの遭遇 | 1D6 | 1 |
| バケネズミが実は「旧人類」であると知る | 1D10 | 1D6 |
| スクィーラの安楽死（選択した場合） | 1D4 | 1 |

**SANが20以下になった場合**: 「不定の狂気（indefinite insanity）」ルール適用。
ただし「業魔化」のフレーバーを使い、呪力が微細に漏れ出す症状として演出する（ゲーム的には通常の狂気と同じ処理）。

## 8. ハンドアウト（HO）テンプレート

### HO1 — 渡辺早季（わたなべ さき）

**年齢**: 12歳（第1話・第2話）/ 26歳（第3話）
**呪力形態**: 視覚共有（離れた場所の景色を見る）+ 念動力

**秘密（PLのみ知る）**:
「姉がいた」という感覚がある。でも誰もそんな姉を知らないし、家族の写真にも姉は映っていない。
夢の中で、誰かが拙い文字で「ごめんなさい」と書くのを見た。

**動機**: 仲間を守りたい。でも何かが、ずっと引っかかっている。

**SANボーナス**: 「仲間が無事だと確認できたとき」SAN +1D3

---

### HO2 — 朝比奈覚（あさひな さとる）

**年齢**: 12歳（第1話・第2話）/ 26歳（第3話）
**呪力形態**: 鏡面（あらゆる光・エネルギーを反射させる面を作る）

**秘密（PLのみ知る）**:
「俺は嘘をついてでも生き残るタイプだ」と自分でわかっている。
でも瞬（青沼瞬）だけは、どうしても見捨てられない気がしている。

**動機**: 生存。でも仲間がいないと「生きていても意味がない」と感じてしまう。

---

### HO3 — 秋月真理亜（あきづき まりあ）

**年齢**: 12歳（第1話）/ 14歳（第2話・途中離脱）
**呪力形態**: 空中浮遊（自分と他者を空中に浮かせる）

**秘密（PLのみ知る）**:
守が「処分対象」だと知っている。彼を見捨てるくらいなら、この町を捨てる。
第2話中盤に「守と共に町を出る」というイベントが確定している。
（PLには事前に「第2話途中でキャラが離脱する」と告知する）

---

### HO4 — 伊東守（いとう まもる）

**年齢**: 12歳（第1話）/ 14歳（第2話・途中離脱）
**呪力形態**: 微弱（呪力レベルが低い。POWは他PCの半分）

**秘密（PLのみ知る）**:
自分が「処分対象」に近いことを、薄々感じている。
夢に不浄猫が出てくる。怖くて怖くてたまらないが、誰にも言えない。
（PLには「第2話途中でキャラが離脱する」と事前に告知する）

---

### HO5 — 青沼瞬（あおぬま しゅん）

**呪力形態**: 全形態（最も優れた呪力使い）

**秘密（PLのみ知る）**:
最近、飼い犬のすばるの形が少し歪んで見える。自分の呪力が漏れているのかもしれない。
もしそれが本当なら、自分がどうなるかわかっている。
（PLには「第2話序盤でキャラが業魔化・消滅する」と事前に告知する。その後、別のキャラを担当）

## 9. GMスクリーン用チートシート

### 愧死機構 早見

```
人間に呪力攻撃 → POW×5判定
  失敗: HP -2D10（即死あり）
  成功: SAN -1D4のみ
  ファンブル: 即死
```

### 呪力使用 早見

```
呪力発動 → POW×5判定（呪力制御でBD追加可）
  成功: 効果発動
  失敗: MP -1、効果なし
  ファンブル: 暴走
  真言あり: ボーナスダイス+1
```

### 業魔エリア 早見

```
エリア進入 → SAN -1D6/1
1R留まる → CON×5。失敗: HP -1D4
近接距離 → CON×3。失敗: HP -1D6
```

### SAN回復（本シナリオ独自）

| 状況 | 回復量 |
|---|---|
| 仲間と安全な場所で一夜を過ごす | +1D3 |
| 真実を知り「それでも生きると決める」 | +1D6（KP判断） |
| 呪力で誰かを救った実感 | +1D3 |
